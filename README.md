# Codemotion Madrid 2025 — Day 2 Recap

Presentación en [Marp](https://marp.app/) con el resumen del día 2 de Codemotion Madrid 2025.

## Temas

- **Desafíos de uso de la IA** — riesgos de la adopción masiva, deepfakes y comunicación humana
- **Migración GitLab → GitHub** — complejidad real de una migración de plataforma
- **Modernización de COBOL** — el abismo entre paradigmas y el futuro del lenguaje
- **Novedades del ecosistema** — Claude Opus 4.7, GPT-5, el filtrado de Claude Code y Knowledge Bases de Karpathy

## Uso local

```bash
# Preview en vivo
marp --watch --server --html slides.md

# Exportar HTML
marp slides.md -o index.html --html --allow-local-files

# Exportar PDF
marp slides.md -o slides.pdf --html --allow-local-files
```
